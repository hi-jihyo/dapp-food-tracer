# DApp : Food Tracer
A blockchain-based food tracer
## Installation
1. Clone the project.
    ```shell
    git clone https://github.com/hi-jihyo/dapp-food-tracer.git
    ```
2. Install npm modules.
    ```shell
    npm install
    ````
2. Run the project.
    ```shell
    npm start
    ````
5. Go to `localhost:8080`.
